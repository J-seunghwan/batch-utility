# requirement
* PySide6 - 6.8.3

# usage
* 특정 경로에 존재하는 폴더 혹은 파일을 일괄적으로 다루기 위한 프로그램
* 폴더들 이름의 특정 패턴 표현 없애기
  - python의 정규식으로 작성해야함
* 폴더들 이름 특정 텍스트 바꾸기
  - [바꾸려는 표현] => [새로운 표현]
 
# caution
* 현재 빈 폴더 삭제만 동작
  - 압축 파일 삭제는 안됨
* 특정 패턴 표현 없애기
  - 현재는 기본적으로 적용되있는 표현만 가능함
